## 技术栈

- **静态站点生成器**: [Hugo](https://gohugo.io/)
- **主题**: [PaperMod](https://github.com/adityatelange/hugo-PaperMod)
- **部署**: GitHub Pages (GitHub Actions)

## 本地开发

```bash
# 克隆仓库
git clone --recursive https://github.com/wangdong/wangdong.github.io.git

# 启动本地服务器
hugo server

# 构建静态文件
hugo
```
